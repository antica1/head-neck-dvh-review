# 头颈肿瘤放疗计划 DVH 快速审核

## 物理师和住院医生的计划审核卡片

> 本 Skill 专用于头颈肿瘤放疗（鼻咽癌、口腔癌、口咽癌、喉癌、下咽癌、唾液腺肿瘤等）。
> OAR 约束和裁决逻辑均基于头颈部位——**不适用于胸/腹/盆部肿瘤，不适用于儿童肿瘤。**

---

## 一、五步审核法

拿到一份 DVH 报告，按以下顺序走——不跳步。

> **⚠️ 重要前提**：所有 OAR 限量值均来自 QUANTEC 等综述，其原始数据多为**动物模型外推**，缺乏人体前瞻性验证。脊髓 ≤45 Gy = 5% 的概率发生脱髓鞘病变，不是"45 Gy 以下安全、45.1 Gy 就出事"。这些数字是**群体概率**，不是**个体阈值**。审核计划时需理解这个本质。

### 第 1 步：硬约束——"一票否决"

| OAR | 约束（常规 30-33fx） | 3 次 SBRT | 5 次 SBRT | 超了的含义 | 超了 → 怎么办 |
|-----|-------------------|----------|----------|-----------|-------------|
| **脑干** | Dmax ≤ 54 Gy | Dmax ≤ 18-21 Gy | Dmax ≤ 23-25 Gy | 动物模型，>54 Gy → 脑干坏死风险不可接受 | ❌ 退回重做 |
| **脊髓** | Dmax ≤ 45 Gy | Dmax ≤ 18-21 Gy | Dmax ≤ 23-25 Gy | 动物模型，45 Gy ≈ 5% 脱髓鞘概率 | ❌ 退回重做 |
| **视交叉** | Dmax ≤ 54 Gy | D0.03cc ≤ 15-17 Gy | D0.03cc ≤ 20-23 Gy | >55 Gy → RION 风险 3-7% | ❌ 退回重做 |
| **视神经** | Dmax ≤ 54 Gy | 同上 | 同上 | 同上 | ❌ 退回重做 |
| **晶状体** | Dmax ≤ 10 Gy | — | — | >4-5 Gy 白内障风险显著升高 | ❌ 退回重做 |

> SBRT 限量来源：AAPM TG-101 / UK SABR 2022 (PMID:35272913)。**SBRT 的 BED 公式在每次 >6 Gy 时失真，不能简单用常规分割的 BED 反推 SBRT 限量。**

> **SBRT 特殊情况——增敏剂修正**：ADC/靶向增敏时，有效剂量 = 物理剂量 × SER（通常 1.2-1.5）。3 周间隔方案提供了额外的晚反应修复窗口——这是常规 SBRT 没有的保护因子。多个混合方案的完整 BED 速算表和裁决流程图见 **`SBRT与常规分割混合方案累积BED计算手册.md`**。

> **这五条列为"硬约束"不是因为它们有绝对的人体证据，而是它们的后果（瘫痪、失明）不可接受。但数值本身带有不确定性。**

### 第 1B 步：约束修正因子——"一成不变的数字不是临床"

QUANTEC 数据基于动物模型的全器官全横断位 2 Gy/fx 照射。实际头颈 IMRT 有四个维度未被 QUANTEC 覆盖：

| 修正因子 | 条件 | 修正方向 | 幅度 | 示例 |
|---------|------|---------|------|------|
| **分次剂量 <2 Gy** | OAR 实际单次剂量低于 2 Gy/fx | **约束可放宽** | BED 等效换算 | 脊髓 Dmax 47 Gy @1.67 Gy/fx → BED 等效 2 Gy/fx 的 43 Gy |
| **部分容积** | 仅 D0.1cc 超标，非 D2cc 超标 | **约束可放宽** | Dmax 50 Gy 仅 0.2% 脊髓病变风险（QUANTEC） | |
| **根治性+年轻** | 生存获益 >> OAR 损伤风险 | **可适当放宽** | ±3-5 Gy | |
| **姑息/老年/合并症** | OAR 损伤风险 >> 生存获益 | **应收紧** | ±3-5 Gy | |
| **再程照射** | 有既往 RT 史 | **必须收紧** | 累积 BED<100 Gy₂ | |
| **部分容积（脑干前表面）** | 仅脑干前缘受照 | **约束可放宽** | 部分 > 全容积 | |

### 第 2 步：靶区覆盖——"够不够"

| 指标 | 合格线 | 不达标 → 怎么办 |
|------|--------|---------------|
| PTV D98% | ≥ 95% 处方剂量 | ⚠️ 优先重优化 |
| PTV D2% | ≤ 107% 处方剂量 | ⚠️ 查热点位置 |
| PTV V95% | ≥ 95% | ⚠️ 查冷点位置 |

> **D98% < 93% → 靶区覆盖不足，先改这个。**

### 第 3 步：软约束——"尽量"

| OAR | 约束 | 超了 → 怎么办 |
|-----|------|-------------|
| 对侧腮腺 Dmean | ≤ 26 Gy | 🟡 超了但不是一票否决。看第5步 |
| 患侧腮腺 Dmean | ≤ 30 Gy | 🟡 |
| 下颌骨 Dmax | ≤ 70 Gy | 🟡 |
| 耳蜗 Dmean | ≤ 45 Gy | 🟡 |
| 咽缩肌 Dmean | ≤ 50 Gy | 🟡 |
| 泪腺 Dmean | ≤ 25 Gy | 🟡 |
| 口腔 Dmean | ≤ 40 Gy | 🟢 |
| 喉 Dmean | ≤ 45 Gy | 🟢 |

### 第 4 步：修正区——"还能不能更好"

| 问题 | 判断 | 行动 |
|------|------|------|
| 热点在哪？ | 在 PTV 内 → 可接受。在 OAR 内 → 优化。在正常组织 → 需要改 | |
| 冷点在哪？ | 在 PTV 边缘 → 可接受。在 GTV 内 → 不可接受 | |
| DVH 曲线顺滑吗？ | 阶梯状 → 多叶光栅或优化问题 | 让物理师检查 |
| 几个弧？ | 单弧 OAR 剂量高 → 加一个非共面弧 | |

### 第 5 步：冲突裁决——"靶区 vs 正常组织"

> **这是最难的。** 靶区覆盖和 OAR 保护同时不达标，怎么选？

| 情景 | 裁决 | 原因 |
|------|------|------|
| **PTV D98% 不达标** vs 下颌骨 Dmax 超标 | **保护靶区** | 下颌骨 ORN 有补救；复发没有 |
| PTV D98% 不达标 vs **脑干/脊髓 Dmax 超标** | **保护 OAR** | 瘫痪比复发更可怕 |
| 对侧腮腺 Dmean 超标 vs 靶区勉强达标 | **接受腮腺超标** | 口干可控；靶区不够不行 |
| 咽缩肌 Dmean 超标 vs 靶区边缘略不足 | **保护靶区** | 吞咽困难有康复；复发没有 |
| 视交叉 Dmax 轻微超标（55-56 Gy）vs 靶区充分 | **接受**——但**必须记录** | NPC 侵犯海绵窦时有时无法避免 |

> **裁决铁律**：
> 硬约束 > 靶区覆盖 > 软约束 > 优化目标
> "瘫痪不能接受。口干可以。复发不行。"

## 三、计划审核报告的双轨制——给物理师 + 给医生

拿到 DVH 数据后，输出的报告分两轨：**一轨给物理师改计划，一轨给医生做临床决策。**

---

### 第一轨：给物理师——"哪里不对、怎么改"

#### 1.1 靶区覆盖缺失分析

| 缺失位置 | 原因 | 严重度 | 告诉物理师什么 |
|---------|------|--------|--------------|
| **体表边缘** | PTV 扩展到皮肤外 → TPS 自动出界 | 🟡 可接受，3-5 mm 内在预期内 | 不需改，记录 |
| **空腔表面（鼻咽/口咽/喉）** | PTV 进入空气腔 → 剂量跌落 | 🟡 可接受，空腔无肿瘤 | 不需改，但注明 D98% 是在空腔层面 |
| **骨-软组织界面** | PTV 跨骨质→剂量重建有误差 | 🟠 需核查——是真低还是重建低？ | 用 Dmean 辅助判断 |
| **靶区缩窄处** | 靶区在 OAR 之间被"挤压" | 🔴 需要重新优化或接受 OAR 超标 | 报："PTV 覆盖不足发生在 X 层面，因 OAR Y 约束过紧" |
| **多靶区交界** | SIB 计划中高剂量区覆盖了低剂量区边缘 | 🟡 确认低剂量区是否真正需要 V100% | 核对处方 |

#### 1.2 剂量拖尾效应

| 位置 | 原因 | 意见 |
|------|------|------|
| 高 Z 材料旁（牙齿/修复体） | 背散射/衰减效应 | 检查是否是计算误差 |
| 靶区远端 | 最后一个弧收束不够 | 建议加一个非共面弧 |
| 皮肤表面（大角度切线入射） | 建成效应 | 加 bolus 或接受 |
| 两个靶区之间 | 多弧交汇处剂量叠加 | 热点管理 |

#### 1.3 OAR 超标——提示修复方案

| OAR | 超标量 | 可能原因 | 建议修复 |
|-----|--------|---------|---------|
| 腮腺 Dmean 超 | +5 Gy | 靶区包裹腮腺深叶 | ① 对侧腮腺造 OAR ring ② 加非共面弧 ③ 接受 |
| 脑干/脊髓 Dmax | +2-3 Gy | 靶区紧贴 | ① 牺牲靶区边缘覆盖 ② BED 修正后判断是否真的超标 |
| 下颌骨 Dmax 超 | +5 Gy | 靶区紧贴骨面 | ① 加下颌骨 PRV ② 收紧热点约束 |
| 晶状体 Dmax 超 | +2 Gy | 前部靶区太靠前 | ① 减前界靶区 ② 加晶状体规避结构 |

### 第二轨：给医生——"怎么选、怎么判"

#### 2.1 多方案对比决策表

当物理师给出了 ≥2 个方案时：

```
方案对比：
┌──────────────┬──────────┬──────────┬──────────┐
│              │ 方案 A   │ 方案 B   │ 方案 C   │
├──────────────┼──────────┼──────────┼──────────┤
│ PTV D98%     │ 95.2% ✓ │ 93.1% ✗  │ 94.8% ⚠  │
│ GTV D98%     │ 98.5% ✓ │ 99.1% ✓  │ 97.2% ⚠  │
│ 脑干 Dmax    │ 53 Gy ✓  │ 55 Gy ⚠  │ 50 Gy ✓  │
│ 脊髓 Dmax    │ 43 Gy ✓  │ 42 Gy ✓  │ 44 Gy ✓  │
│ 腮腺 Dmean   │ 24 Gy ✓  │ 28 Gy ⚠  │ 22 Gy ✓  │
│ HI           │ 0.08 ✓   │ 0.12 ⚠  │ 0.10 ✓   │
│ CI           │ 1.15 ✓   │ 1.20 ⚠  │ 1.10 ✓   │
│ 净优势        │ 0 缺项   │ 1 缺项   │ 1 缺项   │
└──────────────┴──────────┴──────────┴──────────┘

推荐：方案 A → 全部达标。
备选：方案 C → PTV 轻微不足（94.8%）但 OAR 最优——如果 PTV 不足发生在空腔/体表可选。
方案 B → 脑干 55 Gy 需个案判断（见 2.2）。
```

#### 2.2 常见临床困境的裁决指南

**困境 1：PTV D98% = 95%，但脑干有一段 Dmax = 55 Gy——可以接受吗？**

```
判断链：
① 脑干 55 Gy → 软约束超标
② 修正：是部分容积（仅前表面）？是（NPC 常见）→ 风险降低
③ 修正：分次剂量 <2 Gy？30fx → 1.83 Gy/fx → BED 等效 52.7 Gy → 实际低于 54 Gy
④ 年龄：年轻根治 → 可接受更高风险
⑤ 结论：可以接受。记录"脑干前表面部分容积，BED 等效 52.7 Gy"。
   随访时关注但无需退回。
```

**困境 2：GTV V95% = 98%（缺 1%），可以接受吗？**

```
判断链：
① 98% → 轻微不达标。缺的 2% 在哪？
   - 在 GTV 边缘 → 接受，5 mm 微小边缘
   - 在 GTV 中心 → 不可接受，需要确认是否剂量重建问题
   - 在空腔/体表 → 接受
② GTV D98% 是否达标？若无 → 可接受
③ 结论：如果缺失在边缘、空腔或体表 → 可接受。如果缺失在中心 → 退回。
```

**困境 3：GTV V95% 仅 93%——下限在哪？**

```
容忍度层级：
  GTV V95% ≥ 98% → 理想
  GTV V95% 95-97% → 可接受（需记录缺失位置）
  GTV V95% 93-94% → ⚠ 需优化——优先改
  GTV V95% < 93% → 不可接受，退回重做

GTV 和 PTV 的容忍不同：
  GTV 是"肿瘤本身"→ 容忍度极低
  PTV 是"GTV+摆位误差"→ 容忍度稍高（体表/空腔边缘可接受缺失）
```

**困境 4：腮腺 Dmean 比 PTV 覆盖重要，还是反过来？**

```
裁决：
  腮腺 Dmean 超标 ≤ 3 Gy → 优先靶区覆盖
  腮腺 Dmean 超标 > 5 Gy（> 31 Gy）→ 再优化，因为不可逆口干

原则：对侧腮腺优先保护；患侧可适当牺牲。
     口干有 Pilocarpine / 针灸等补救，复发没有。
```

## 四、常见失败模式及修复建议

| 失败 | 原因 | 告诉物理师怎么改 |
|------|------|----------------|
| **脑干超标** | 靶区太靠近脑干 | ① 牺牲靶区边缘覆盖 ② 考虑加一挡低危CTV ③ 质子 |

## 六、特殊情况：ADC 停药后的剂量补偿

### 问题

ADC 靶向药物因耐受差停了。后续几程不能再联合 ADC——仅靠放疗，需要增加每次照射多少 Gy 才能达到等效的肿瘤杀伤？

### 核心公式

目标：补齐缺掉的部分。

```
SER = 增敏比（1.2-1.5）
已完成 N 程（有 ADC），剩余 M 程（无 ADC）

需要补齐的 BED₁₀ = M × (3.5×SER×(1+3.5×SER/10) - 3.5×(1+3.5/10))
```

| SER | 每程有 ADC 的 BED₁₀ | 每程无 ADC 的 BED₁₀ | **差值 = 需要补齐** |
|-----|-------------------|-------------------|------------------|
| 1.2 | 5.96 Gy₁₀ | 4.73 Gy₁₀ | **1.24 Gy₁₀ / 程** |
| 1.5 | 8.01 Gy₁₀ | 4.73 Gy₁₀ | **3.28 Gy₁₀ / 程** |

> 纯物理剂量 3.5 Gy 的 BED₁₀ = 3.5×(1+3.5/10) = 4.73 Gy₁₀。ADC 增敏后提升到 5.96-8.01 Gy₁₀——即 ADC 每程"免费"提供了 26-69% 的额外肿瘤杀伤。

### 补偿方案

假设 ADC 在 **第 3 程后** 停药——即已完成 3 程（有 ADC），剩余 3 程（无 ADC）：

| 参数 | SER=1.2 | SER=1.5 |
|------|---------|---------|
| 需要补齐总量 | 3 × 1.24 = **3.7 Gy₁₀** | 3 × 3.28 = **9.8 Gy₁₀** |
| 剩余每程需要 BED₁₀ | 4.73 + 3.7/3 = **5.96 Gy₁₀** | 4.73 + 9.8/3 = **8.00 Gy₁₀** |
| 剩余每程需要物理剂量 | X×(1+X/10)=5.96 → **X ≈ 4.2 Gy** | X×(1+X/10)=8.00 → **X ≈ 5.0 Gy** |
| 脊髓单次限量 | TG-101: 10-14 Gy | 4.2/5.0 → ✅ 安全 |
| 6 程总晚期等效 | 3×3.5 + 3×4.2 → BED₃=58 → **等效 35 Gy** | 3×3.5 + 3×5.0 → BED₃=68 → **等效 41 Gy** |
| 脊髓安全 | ✅ 35 < 45 | ✅ 41 < 45 |

假设 ADC 在 **第 1 程后** 即停药：

| 参数 | SER=1.2 | SER=1.5 |
|------|---------|---------|
| 剩余每程需要 | **X ≈ 4.2 Gy**（同上——每程需补齐的量不变） | **X ≈ 5.0 Gy** |
| 脊髓安全 | BED₃ = 3.5 + 5×4.2 → 等效 **37 Gy** ✅ | 3.5 + 5×5.0 → 等效 **43 Gy** ⚠ 接近底线 |

假设 ADC **从未使用**——纯放疗 6 程全补到等效剂量：

| 参数 | 补到 SER=1.2 等效 | 补到 SER=1.5 等效 |
|------|------------------|------------------|
| 每程需要 | **X ≈ 4.2 Gy** | **X ≈ 5.0 Gy** |
| 6 程总晚期等效 | **等效 37 Gy** ✅ | **等效 43 Gy** ⚠ |
| 总物理剂量 | 6 × 4.2 = 25.2 Gy | 6 × 5.0 = 30 Gy |
| 脊髓 Dmax/次 | 4.2 Gy（<10 Gy TG-101）✅ | 5.0 Gy（<10 Gy）✅ |

---

### 结论

| 问题 | 答案 |
|------|------|
| ADC 停药后需要增加多少 Gy？ | **每程从 3.5 Gy 提升至 4.2-5.0 Gy**（取决于 SER 和已完成的 ADC 程数） |
| 脊髓是否安全？ | 单次 4.2-5.0 Gy 远低于 TG-101 限量（10-14 Gy）。总晚期等效 35-43 Gy 低于 45 Gy |
| 能否纯放疗替代全 6 程？ | 可以。4.2 Gy × 6（25.2 Gy 总量）→ 脊髓安全。但 SER=1.5 等效时脊髓接近底线 |
| 给物理师的指令 | "ADC 停药。剩余 X 程每次提升至 Y Gy。保持脊髓 Dmax < Y Gy 且总后期等效 <45 Gy" |

### OAR 约束的证据来源

| 约束 | 原始证据 | 有多少把握 |
|------|---------|-----------|
| 脊髓 ≤45 Gy | 大鼠/豚鼠模型 → 脱髓鞘概率曲线 → 外推人类 | ⚠️ 没有人体实验数据 |
| 脑干 ≤54 Gy | 动物模型 + 回顾性病例系列 | ⚠️ |
| 视神经 ≤55 Gy | QUANTEC 汇总：多中心回顾性，非前瞻性 | ⚠️ |
| 腮腺 Dmean ≤26 Gy | **有临床数据**——多个 IMRT 系列测量口干与剂量的关系 | ✅ 有临床证据 |
| 下颌骨 Dmax ≤70 Gy | 临床回顾，但 ORN 还有感染/创伤等其他因素 | ⚠️ 有混杂 |
| 晶状体 ≤10 Gy | 临床数据——白内障风险与剂量的剂量-效应曲线较清晰 | ✅ 有临床证据 |

### 这意味着什么

- **神经 OAR 限量（脊髓/脑干/视交叉/视神经）**：理论外推，存在较大不确定性。54 Gy 可能对某些个体已经危险，55 Gy 可能对另一些个体安全。
- **腮腺/晶状体限量**：有较好的临床证据基础，可信度较高。
- **对"硬约束"的正确态度**：不是"超过 0.1 Gy 必出事"，而是"超了以后风险概率从群体可接受水平上升到了需要个案权衡的水平"。
