# SBRT与常规分割混合方案累积BED计算手册

> 关联 Skill: `head-neck-dvh-plan-review` | 朱国培 · 上海九院

## 核心公式

BED = nd × (1 + d/(α/β))
EQD2 = BED / (1 + 2/(α/β))

| 组织 | α/β |
|------|------|
| 肿瘤/急性反应 | 10 Gy |
| 晚反应正常组织 | 3 Gy |

## 速算表

### ADC/IO 增敏后的有效 SBRT

> SER 1.2-1.5 来自顺铂 III 期 + DNA 修复抑制剂临床前数据的外推。ADC 无临床 SER 数据。

### Quad-Shot 方案 (JAMA Otolaryngol 2026, PMID:42313403)

3.7 Gy × 4 BID Q4w × 3 程 + Pembrolizumab
每程 BED₁₀ = 20.3 Gy₁₀（肿瘤）/ BED₃ = 33.1 Gy₃（晚反应）

### Q3w 单次方案 (九院经验)

3.5 Gy × 1 Q3w × 6 程 + IO/ADC
每程 BED₁₀ = 4.73 Gy₁₀ / BED₃ = 7.58 Gy₃

## 免疫阈值

| 单次剂量 | ICD 效果 |
|---------|---------|
| <4 Gy | 可能未达阈值 |
| 4-6 Gy | 弱-中等 |
| **6-12 Gy** | **最佳** |
| >12 Gy | 可能杀伤 T 细胞 |
