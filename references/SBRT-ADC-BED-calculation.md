# SBRT + ADC 放射增敏——复发头颈肿瘤的 BED 计算

## 临床情景

复发头颈肿瘤，不可手术。SBRT 3.5 Gy × 6 程（每 3 周一次）+ ADC 靶向药物（照射后 24 小时内用药）。

## 关键参数

| 参数 | 值 |
|------|-----|
| 每次物理剂量 | 3.5 Gy |
| 间隔 | 3 周 |
| 疗程数 | 6 |
| 总物理剂量 | 21 Gy |
| ADC 增敏比 (SER) | 1.2-1.5 |
| 肿瘤 α/β | 10 Gy |
| 晚反应 α/β | 3 Gy |

## BED 核心公式

- BED = nd × (1 + d/(α/β))
- 等效 2 Gy/fx = BED / (1 + 2/(α/β))

## 三种必须考虑的修正

1. **3 周间隔→正常组织修复**：晚反应 BED 因充分修复而低于连续照射
2. **3 周间隔→肿瘤再增殖**：18 周总时间→约损失 8 Gy₁₀
3. **ADC 增敏**：有效剂量 = 物理剂量 × SER

## 结论

- 纯物理剂量 3.5 Gy × 6 → 等效仅 ~17 Gy（2 Gy/fx），完全不够杀死复发肿瘤
- ADC 增敏（SER=1.2-1.5）→ 等效 ~23-34 Gy，SER ≥1.5 才可靠
- ADC 停药后：每程需提升至 4.2-5.0 Gy 以补偿
- 脊髓安全：单次 4.2-5.0 Gy 远低于 TG-101 限量（10-14 Gy）
